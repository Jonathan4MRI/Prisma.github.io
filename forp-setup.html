The 932 fORP System

Created by Chad Smiddy, last modified by Pierce, Elizabeth on May 01, 2023

The 932 fORP System consists of Handheld Devices, a Fiber Optic Bundle, and an Interface Unit. Contact staff for setting up new projects.

Figure 1
Figure 2
Figure 3

Function:

Records subject responses while they are inside the MR scanner. It may also be used for testing and training in the MR console area prior to scanning.

Location:

The Handheld Button Box and Joystick is located in the MRI scanner room and their cable and the Fiber Optic Bundle may be brought out into the MR console area for practice and troubleshooting. The Interface Unit is in the Console room and is not permitted inside the MR scanner's room.

Setup:

Selecting Mode:

Figure 4

Description:
Use the knob on the front Panel of the 932 Interface Unit to select the MODE appropriate to your setup. Pushing the knob locks in the response and rotating the knob navigates the menu. To select the Mode perform the following:

  • Connect the device you'd like to use: Button Box or Joystick
  • The fORP should be left in the Default state which is HID NAR 12345 for the Button Box. The Button Box is usually connected.
  • Push the knob IN and select YES
  • Scroll down and select AUTOCONFIGURE
  • Scroll down and select HID NAR 12345.

How to Reconfigure the fORP for the Button Box from the Joystick Mode:

These settings (FIU-005 HID NAR 12345) are for the Joystick so if you use the Button Box you must reconfigure the fORP.

Figure 5
  • Push in the button on the fORP, then select Yes.
Figure 6
  • With the device you'd like to use connected, select: AUTOCONFIGURE
Figure 7
  • Scroll the knob to select HID NAR 12345 (the Trigger is a 5), with the Button Box connected you will see the Device Code (HHSC-1x4-L) on the fORP display.

The Button Box has a sticker on it with this Device Code; shown here.

Figure 8

Monitoring Responses:

You can use the LED Array on the front panel of the 932 Interface Unit to monitor subject's responses and the scanner trigger pulse.

Connections:

Figure 9
  • The handheld device is connected to the fiber optic bundle.
  • The fiber optic bundle runs through the front waveguide and is connected to the Optical Port (P4) on the 932 Interface.
  • The 932 Interface Unit is connected to the presentation computer via a USB cable (P3) through a USB Hub and a KVM Switch.
  • The BNC cable is connected to the TTL TRIGGER Port (P6).
