Overview

The 932 fORP System consists of Handheld Devices, a Fiber Optic Bundle, and an Interface Unit. This system allows researchers to record subject responses while inside the MR scanner.

Function & Location

Function

Records subject responses while they are inside the MR scanner. It may also be used for testing and training in the MR console area prior to scanning.

Location

  • The Handheld Button Box and Joystick are located in the MRI scanner room
  • Their cable and the Fiber Optic Bundle may be brought out into the MR console area for practice and troubleshooting
  • The Interface Unit is in the Console room and is NOT permitted inside the MR scanner's room

Selecting Mode

fORP Interface Unit - Front Panel Controls

üéõÔ∏è Control Knob Operation:
Use the knob on the front Panel of the 932 Interface Unit to select the MODE appropriate to your setup.
  • Pushing the knob locks in the response
  • Rotating the knob navigates the menu

To Select the Mode:

  • Connect the device you'd like to use: Button Box or Joystick
  • The fORP should be left in the Default state which is HID NAR 12345 for the Button Box. The Button Box is usually connected.
  • Push the knob IN and select YES
  • Scroll down and select AUTOCONFIGURE
  • Scroll down and select HID NAR 12345

Monitoring Responses

You can use the LED Array on the front panel of the 932 Interface Unit to monitor subject's responses and the scanner trigger pulse in real-time.

üí° Pro Tip:
The LED indicators provide immediate visual feedback, making it easy to verify that the system is properly receiving and processing responses during setup and testing.

Connections

Complete fORP System Connection Diagram

Connection Details:

Step 1 The handheld device is connected to the fiber optic bundle
Step 2 The fiber optic bundle runs through the front waveguide and is connected to the Optical Port (P4) on the 932 Interface
Step 3 The 932 Interface Unit is connected to the presentation computer via a USB cable (P3) through a USB Hub and a KVM Switch
Step 4 The BNC cable is connected to the TTL TRIGGER Port (P6)
